AI File Organizer is a desktop application developed using Python and PyQt6. It helps users organize files automatically by scanning a selected folder and sorting files into categorized directories based on their file extensions.
The application is designed using the MVC (Model-View-Controller) architecture, separating the user interface, application logic, and file processing services to improve code maintainability and scalability.

🔍 Scan folders and analyze file types
📊 Display file statistics
